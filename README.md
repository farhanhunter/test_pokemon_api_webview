# Pokemon List Webview Thymeleaf (Spring Boot)

## Deskripsi Aplikasi

Aplikasi backend berbasis **Spring Boot** untuk webview
- **Client Side** : Thymeleaf

## Fitur Utama

- List Pokemon
- Relational Data
- Sorting

## How to Run
- Clone Repository
- Run `mvn spring-boot:run`
- Open Browser `http://localhost:5200/poke/list`
- Make Sure you have internet connection for building dependencies
- Make Sure you have java 17 installed
